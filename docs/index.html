Secure Your Ports with PortSentry

A premium, responsive Windows desktop dashboard to monitor local services, inspect CPU & memory telemetry, and terminate entire process trees in one click.

Download Setup Installer Download Portable App

Core Capabilities

Real-Time Discovery

Scans and groups active port listeners (Dev servers, Database services, local Web hosts) automatically with custom classification categories.

Process Tree Kill

Terminates the entire process tree on Windows. Sub-worker nodes (like Vite, Next.js, Webpack) are cleanly shut down, preventing locked ports.

System Alerts

Uses native system notifications to immediately alert you whenever a new local development environment or server launches.

Minimize-to-Tray

Double-clicking or closing the window parks it quietly inside your system notification tray area, keeping active port status just a hover away.

Quick Start Installation

Get PortSentry up and running locally in less than a minute. Clone the codebase, install standard dependencies, and execute the boot script.

# 1. Clone repository
git clone https://github.com/jojin1709/PortSentry.git
cd PortSentry
# 2. Install dependencies
npm install
# 3. Boot app locally
npm start

Packaging Portable Binary

Package the application codebase into a single-file standalone installer `portsentry 1.0.0.exe` targeting 64-bit Windows architecture.

# Build stand-alone executable
npm run build

The builder output will be located in the newly created `dist/` directory.
